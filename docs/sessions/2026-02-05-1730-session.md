## Decisions Made

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Color mode | Dark only | E-ink aesthetic, reduced eye strain, crypto standard |
| Base palette | Monochrome grays | #0A0A0A → #E8E8E8, no pure black/white |
| Semantic colors | Muted green/red only | Reserved for bullish/bearish signals |
| Typography | Geist + Geist Mono | Vercel's typeface, small-size legibility |
| Data display | Monospace numbers | Vertical alignment, terminal aesthetic |
| UI density | Compact (40px rows) | Pro-trader feel, more data per viewport |
| Animation | Near-zero | E-ink aesthetic, instant state changes |
| Component library | shadcn/ui | Tailwind-native, RSC compatible |
